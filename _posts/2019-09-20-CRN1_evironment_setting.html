1.1 环境配置

1.1.1 安装 Python3 和 pip3:

{% highlight Bash %} sudo apt-get install -y python3-dev build-essential libssl-dev sudo apt-get install -y libffi-dev libxml2 libxml2-dev sudo apt-get install -y libxslt1-dev zlib1g-dev libcurl4-openssl-dev sudo apt-get install -y python3 sudo apt-get install -y python3-pip {% endhighlight %}

1.1.2 安装请求库 requests、Selenium 和 aiohttp

{% highlight Bash %} pip3 install requests selenium aiohttp {% endhighlight %}
